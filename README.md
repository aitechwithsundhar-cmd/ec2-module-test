# EC2 Module Test

Terraform practice project for testing reusable AWS EC2 modules.

---

## Project Overview

This repository demonstrates how to use reusable Terraform modules for deploying AWS EC2 instances.

The project focuses on:
- Terraform module usage
- AWS EC2 provisioning
- Variable management
- Security groups
- Infrastructure as Code (IaC)

---

## Technologies Used

- Terraform
- AWS EC2
- AWS Security Groups

## Prerequisites

Before running this project, make sure you have:

- AWS Account
- AWS CLI configured
- Terraform installed
- IAM permissions for EC2

Check Terraform version:

```bash
terraform version
```

---

## Terraform Commands

### Initialize Terraform

```bash
terraform init
```

### Validate Configuration

```bash
terraform validate
```

### Plan Infrastructure

```bash
terraform plan
```

### Apply Infrastructure

```bash
terraform apply
```

### Destroy Infrastructure

```bash
terraform destroy
```
